# Student Result Management System (SRMS)

In Student result management system, it allows the students to search and view their exam results with their register number. Also, the students can abel to view the various notices and announcements posted by the institution management in the portal. The Admin user has the most priveleges in this system and can add,view,update and delete the students, classes, subjects, subject combinations, declare or edit exam results, post/delete notices.


## Installation
1.Download the zip file
2.Extract the file and copy srms folder
3.Paste it inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)
4.Open PHPMyAdmin (http://localhost/phpmyadmin)
5.Create a database with name `srms`.
6.Import the tables using srms_db.sql file(given inside the zip package in SQL file folder) in database.
7.Visit the URL http://localhost/srms/index.php
